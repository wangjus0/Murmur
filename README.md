## To run locally

Clone the project

```bash
  git clone https://github.com/wangjus0/Murmur.git
```

Go to the project directory

```bash
  cd Murmur
```

Install dependencies

```bash
  npm install
```

Start the electron application

```bash
  npm run dev:electron
```
